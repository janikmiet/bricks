## Installation

```
download.file(paste0("https://github.com/janikmiet/bricks/archive/master.zip"), "bricks.zip")
install.packages("./bricks.zip", repos = NULL, type = "source", lib = .libPaths()[1])
file.remove("./bricks.zip")
```
